## 机器语言
* 机器语言是机器指令的集合
* 机器指令是一台机器可以正确执行的命令
* 电子计算机的机器指令是一列二进制数字
* 计算机将二进制的机器指令转成一列 **高低电平** ，使计算机的电子器件受到驱动，进行计算
* 在计算机中，执行上述指令操作的器件，就是CPU(中央处理器)，是微处理器
* 每一种微处理器需要不同的电平脉冲来控制，因为微处理器的内部结构和硬件设计不同

## 汇编语言的产生
* 程序员 -> 汇编指令 -> 编译器 -> 机器码 -> 计算机
* 汇编指令是机器码便于记忆的格式
```
操作：寄存器BX的内容送到AX中
机器指令： 1000100111011000
汇编指令：mov ax,bx
```
* 寄存器，CPU中可以存储数据的器件
